API ENDPOINTS FOR TESTING
================================================================================

1. Create Account:
   POST http://localhost:8000/api/create-account
   Content-Type: application/json
   Body: {
     "email": "testuser@example.com",
     "password": "Test123!@#",
     "confirm_password": "Test123!@#",
     "first_name": "Test",
     "last_name": "User",
     "gender": "male",
     "birthday": "1990-01-15",
     "street": "123 Test Street",
     "city": "Test City",
     "height": 175,
     "weight": 70
   }

2. Login:
   POST http://localhost:8000/api/login
   Content-Type: application/json
   Body: {
     "email": "testuser@example.com",
     "password": "Test123!@#"
   }
   Response includes JWT token - use this for authenticated requests

3. Get User Profile (requires JWT):
   GET http://localhost:8000/api/users/profile
   Authorization: Bearer <JWT_TOKEN>

4. Update User Profile (requires JWT):
   PUT http://localhost:8000/api/users/profile
   Authorization: Bearer <JWT_TOKEN>
   Content-Type: application/json
   Body: {
     "first_name": "Updated",
     "last_name": "Name",
     ...
   }

  }'
